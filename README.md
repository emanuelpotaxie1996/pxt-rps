# Pedra, paper, tisores

Aquesta extensió per MakeCode micro:bit afegeix un joc de pedra, paper, tisores contra un ordinador; i un joc del mateix però contra una altra micro:bit amb ràdio que sigui a prop seu i que tingui el mateix grup de ràdio que vostè.
Si el link no funciona, copiï tot el contingut de `pps.ts` i creiï un nou arxiu a qualsevol projecte de MakeCode. A aquest arxiu, enganxi el contingut copiat per tenir la extensió.
